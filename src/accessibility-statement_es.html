Declaración de accesibilidad

Esta declaración se aplica a los contenidos publicados en el siguiente dominio: ‘https://europa.eu/eu-for-children/index_es.html ’

No se aplica a otros contenidos o sitios web publicados en sus subdominios. Esos sitios web y su contenido tendrán su propia declaración de accesibilidad específica.

Este sitio web está gestionado por la DG Comunicación. Está diseñado para ser utilizado por el mayor número posible de personas, incluidas las personas con discapacidad.

Debe permitir:

  • ampliar la imagen hasta el 200 % sin problemas
  • navegar por la mayor parte del sitio web utilizando solo el teclado
  • navegar por la mayor parte del sitio web utilizando un sistema moderno de lectores de pantalla y reconocimiento de voz (en un ordenador o teléfono)

Este sitio web está diseñado para cumplir la norma técnica para sitios web y aplicaciones para dispositivos móviles EN 301 549, versión 3.2.1 . Se aproxima al nivel «AA» de las Pautas de Accesibilidad de Contenido Web (WCAG), versión 2.1.

Situación en cuanto al cumplimiento

Este sitio web cumple parcialmente la norma técnica EN 301 549, versión 3.2.1 y las Pautas de Accesibilidad de Contenido Web (WCAG) 2.1, nivel AA. Para más información, véase «Contenido no accesible»».

El sitio web se sometió a prueba por última vez el 31 de octubre de 2024.

Preparación de la declaración

La presente declaración fue revisada el 31 de octubre de 2024.

La declaración se basa en la revisión de una muestra representativa de páginas web por un contratista externo encargado del desarrollo del sitio web, utilizando una combinación de pruebas manuales y automatizadas.

Denos su opinión

Nos interesa su opinión sobre la accesibilidad del sitio web de la publicación para personas que votan por primera vez. Le rogamos que nos comunique si ha encontrado barreras de accesibilidad:

  • Correo electrónico: EU-CHILDPARTICIPATION@ec.europa.eu

Intentamos responder a los comentarios en un plazo de 15 días hábiles a partir de la recepción de la solicitud por el servicio competente de la Comisión.

Compatibilidad con navegadores y tecnologías de apoyo

El sitio web de la publicación para personas que votan por primera vez está diseñado para ser compatible con las siguientes tecnologías de apoyo más utilizadas:

  • la última versión de los navegadores Google Chrome y Apple Safari;
  • en combinación con las últimas versiones de JAWS, NVDA, VoiceOver y TalkBack.

Especificaciones técnicas

La accesibilidad del sitio web de la publicación para personas que votan por primera vez se basa en las siguientes tecnologías para funcionar con la combinación particular de navegador web y tecnologías de apoyo o plugins instalados en su ordenador:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Teclas de acceso

Las teclas de acceso permiten a los usuarios navegar por este sitio web utilizando el teclado. Puede obtener más información sobre teclas de acceso, haga clic en W3C Accessibility Guidelines.

Teclas de acceso disponibles

Este sitio utiliza un conjunto de teclas que coincide en gran parte con la mayoría de recomendaciones internacionales sobre teclas de acceso. La tecla de acceso disponible es la siguiente:

  • L - Cambiar el idioma
