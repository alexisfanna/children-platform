Declaração de acessibilidade

Esta declaração aplica-se aos conteúdos publicados no domínio: ‘https://europa.eu/eu-for-children/index_pt.html ’

Não é aplicável a outros conteúdos ou sítios Web publicados em qualquer dos seus subdomínios. Esses sítios Web e respetivos conteúdos terão a sua própria declaração de acessibilidade específica.

Este sítio Web é gerido pela DG COMM e foi concebido para ser utilizado pelo maior número possível de pessoas, incluindo pessoas com deficiência.

Os utilizadores devem poder:

  • ampliar a imagem até 200 % sem dificuldade,
  • navegar na maior parte do sítio Web utilizando apenas um teclado,
  • navegar na maior parte do sítio Web utilizando um leitor de ecrã moderno e software de reconhecimento de voz (no seu computador ou telefone).

Este sítio Web foi desenvolvido em conformidade com a norma técnica para sítios Web e aplicações móveis EN 301 549, versão 3.1.2. que é equivalente ao nível «AA» das diretrizes sobre a acessibilidade dos conteúdos Web (Web Content Accessibility Guidelines – WCAG), versão 2.1.

Estado de conformidade

Este sítio Web está parcialmente em conformidade com a norma técnica EN 301 549, versão 3.2.1 e com as Diretrizes sobre a acessibilidade dos conteúdos Web (Web Content Accessibility Guidelines – WCAG), versão 2.1, nível «AA». Para mais informações, ver «Conteúdos não acessíveis».

O sítio Web foi testado, pela última vez, em 31 de outubro de 2024.

Elaboração da declaração

A presente declaração foi revista em 31 de outubro de 2024.

Baseia-se na análise de uma amostra representativa de páginas Web, realizada por um contratante externo responsável pelo desenvolvimento do sítio Web, utilizando uma combinação de testes manuais e automatizados.

Comentários

Partilhe a sua opinião sobre a acessibilidade do sítio Web da «Publicação para eleitores que votam pela primeira vez». Contacte-nos se encontrar obstáculos relacionados com a acessibilidade:

  • Endereço de correio eletrónico: EU-CHILDPARTICIPATION@ec.europa.eu

Procuraremos responder-lhe no prazo de 15 dias úteis, a contar a partir da data de receção do seu pedido pelo serviço competente da Comissão.

Compatibilidade com navegadores e tecnologias de apoio

O sítio Web da «Publicação para eleitores que votam pela primeira vez» foi concebido para ser compatível com as seguintes tecnologias de apoio mais comummente utilizadas:

  • as últimas versões dos navegadores Google Chrome e Apple Safari,
  • combinadas com as versões mais recentes das tecnologias JAWS, NVDA, VoiceOver e TalkBack.

Especificações técnicas

Para garantir a acessibilidade e funcionar no contexto específico da utilização combinada de um navegador Web e de tecnologias de apoio ou módulos (plug-ins) instalados no seu computador, o sítio Web da «Publicação para eleitores que votam pela primeira vez» depende das seguintes tecnologias:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Teclas de acesso

As teclas de acesso permitem-lhe navegar neste sítio Web utilizando o teclado. Para obter mais informações sobre teclas de acesso, consulte as Diretrizes sobre a acessibilidade W3C.

Teclas de acesso disponíveis

Este sítio Web utiliza uma configuração muito próxima das principais recomendações internacionais sobre teclas de acesso. A tecla de acesso disponível é:

  • L - Alterar a língua
