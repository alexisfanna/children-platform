Η ΕΚΔΟΣΗ ΓΙΑ ΠΑΙΔΙΑ

Η Ευρωπαϊκή Ένωση (ΕΕ εν συντομία) είναι μια ομάδα 27 συνεργαζόμενων χωρών. Η ΕΕ θέλει να διασφαλίσει ότι τα παιδιά μπορούν να έχουν μια καλή ζωή.

Σύσταση είναι ένα έγγραφο πολιτικής που συμβουλεύει τις ευρωπαϊκές κυβερνήσεις για τις ενέργειες που πρέπει να κάνουν.

Αυτή η σύσταση εξηγεί πώς οι ενήλικες μπορούν να συνεργαστούν καλύτερα για την ασφάλεια των παιδιών.

Αποτελεί μέρος της στρατηγικής της ΕΕ για τα δικαιώματα του παιδιού, η οποία θέτει την καταπολέμηση της βίας κατά των παιδιών και την προστασία τους ως κορυφαία προτεραιότητα για την ίδια την ΕΕ.

Περισσότερα από 1.000 παιδιά εξέφρασαν τις απόψεις κατά την κατάρτιση αυτής της σύστασης μέσω της πλατφόρμα της ΕΕ για τη συμμετοχή των παιδιών.

ΣΤΟΧΟΙ

  • Η προστασία των παιδιών από κάθε μορφή βίας.
  • Η συνεργασία όλων για την ασφάλεια των παιδιών.
  • Η ενθάρρυνση των χωρών της ΕΕ (κυβερνήσεις, περιφέρειες και πόλεις) για τη βελτίωση των κανόνων τους και του τρόπου προστασίας των παιδιών.
  • Η συνδρομή ώστε τα παιδιά να είναι ασφαλή σε όλο τον κόσμο.
  • Η ανάδειξη των τρόπων με τους οποίους οι νόμοι, οι πολιτικές και τα χρήματα της ΕΕ μπορούν να βοηθήσουν τις χώρες της ΕΕ στην προστασία των παιδιών.

ΚΥΡΙΑ ΜΗΝΥΜΑΤΑ

ΑΚΟΥΣΤΕ ΤΑ ΠΑΙΔΙΑ ΚΑΙ ΑΡΧΙΣΤΕ ΑΠΟ ΤΙΣ ΑΝΑΓΚΕΣ ΤΟΥΣ

Οι χώρες της ΕΕ θα πρέπει:

  • να βρουν τρόπους ώστε τα παιδιά να πουν τις απόψεις τους, να ληφθούν σοβαρά υπόψη και να λαμβάνουν σαφείς πληροφορίες.
  • να προωθήσουν την πλατφόρμα της ΕΕ για συμμετοχή των παιδιών προκειμένου να εμπλακούν τα παιδιά και να εκφράσουν την άποψή τους.
  • να προωθήσουν την πλατφόρμα της ΕΕ για συμμετοχή των παιδιών προκειμένου να εμπλακούν τα παιδιά και να εκφράσουν την άποψή τους.
  • να καλύπτουν τις ανάγκες των παιδιών και να εργάζονται για κάθε παιδί, με ιδιαίτερη προσοχή στα παιδιά που διατρέχουν τον μεγαλύτερο κίνδυνο βίας.
  • να προστατεύουν τα παιδιά από κάθε διάκριση λόγω θρησκείας, φύλου, καταγωγής, αναπηρίας, πεποιθήσεων ή γενετήσιου προσανατολισμού.
  • να διασφαλίζουν την ασφάλεια των παιδιών στο σπίτι, στο σχολείο, στον αθλητισμό και σε άλλες δραστηριότητες.
  • να διασφαλίζουν την ασφάλεια των παιδιών στο διαδίκτυο, για παράδειγμα, ενημερώνοντας παιδιά και οικογένειες πώς να παραμένουν ασφαλείς όταν χρησιμοποιούν διαδικτυακά εργαλεία και διασφαλίζοντας ότι το διαδίκτυο είναι ασφαλές για τα παιδιά.

ΠΡΟΣΤΑΣΙΑ ΤΗΣ ΥΓΕΙΑΣ ΤΩΝ ΠΑΙΔΙΩΝ ΚΑΙ ΠΡΟΛΗΨΗ ΤΗΣ ΒΙΑΣ

Οι χώρες της ΕΕ θα πρέπει:

  • να έχουν ένα εθνικό σχέδιο για την υποστήριξη της ψυχικής υγείας των παιδιών.
  • να παρέχουν υπηρεσίες υποστήριξης σε παιδιά με προβλήματα ψυχικής υγείας.
  • να λαμβάνουν μέτρα για την καταπολέμηση όλων των μορφών εκφοβισμού.
  • να υποστηρίζουν την υγεία των παιδιών (π.χ. υγιεινή διατροφή, αθλητισμός, απουσία εθισμών).
  • να λαμβάνουν μέτρα για την πρόληψη της βίας (π.χ. απαγόρευση στους ενήλικες να ασκούν σωματική βία σε παιδιά και παροχή υποστήριξης σε οικογένειες).
  • να διευκρινίσουν πώς τα παιδιά μπορούν να αναφέρουν περιστατικά βίας προκειμένου να λάβουν άμεσα βοήθεια.

ΣΥΝΕΡΓΑΣΙΑ ΓΙΑ ΤΗΝ ΠΡΟΣΤΑΣΙΑ ΤΩΝ ΠΑΙΔΙΩΝ

Οι χώρες της ΕΕ θα πρέπει:

  • hνα έχουν εθνικά σχέδια για την καταπολέμηση της βίας κατά των παιδιών και να διασφαλίζουν ότι υπάρχουν επαρκείς πόροι και προσωπικό για την εφαρμογή τους.
  • εκπαιδευτικοί, ιατρικό προσωπικό, αστυνομία, κοινωνικοί λειτουργοί, δικαστές και άλλοι επαγγελματίες πρέπει να είναι εκπαιδευμένοι και να γνωρίζουν πώς να συνεργάζονται για την προστασία και την υποστήριξη των παιδιών.
  • να ικανοποιούν τις ανάγκες των παιδιών που δεν ζουν με την οικογένειά τους και να φροντίζουν ώστε να ζουν σε μια σταθερή οικογένεια ή κοινότητα.
  • να προσαρμόσουν το σύστημα απονομής δικαιοσύνης στις ανάγκες των παιδιών και να βρουν τρόπους ώστε να διασφαλιστεί ότι όλοι αντιμετωπίζονται δίκαια.
  • να προστατεύουν τα παιδιά που είναι μετανάστες, ιδίως εκείνα που έχουν έρθει χωρίς τις οικογένειές τους.
  • να συλλέγουν καλές πληροφορίες για την προστασία των παιδιών.
  • να χρησιμοποιούν εργαλεία, νόμους, πολιτικές και πόρους της ΕΕ για την καλύτερη προστασία των παιδιών.

ΠΡΟΣΤΑΣΙΑ ΤΩΝ ΠΑΙΔΙΩΝ ΣΕ ΟΛΟ ΤΟΝ ΚΟΣΜΟ

Οι χώρες της ΕΕ θα πρέπει να λάβουν μέτρα για:

  • την προστασία των παιδιών στην πολιτική τους σε όλο τον κόσμο.
  • την καταπολέμηση της παιδικής εργασίας.
  • την προστασία των παιδιών σε ένοπλες συγκρούσεις.
  • την προστασία των παιδιών από τις επιπτώσεις της κλιματικής αλλαγής και των περιβαλλοντικών κινδύνων και την συμμετοχή τους στην εξεύρεση λύσεων.
